# File Upload Web App

This is a simple responsive web application that allows users to upload files to the server and lists uploaded files. Built with Node.js, Express, and Bootstrap.

## Features

- Upload files of any type.
- Stores uploaded files in the `uploads` directory.
- Lists uploaded files with download links.
- Responsive design using Bootstrap 5.

## Setup

```bash
npm install
npm run dev # nodemon
# or
npm start
```

Open your browser at `http://localhost:3000`.
